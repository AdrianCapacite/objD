## Widget A widget is the base element for basically everything in objD. |property | | |--|--| | generate(Context) | returns the underlying Widget Tree | | toMap() | Similar to toString, but returns a Map tree of the child in generate | **Example:** We can use an Widget to get or build functionality. e.g ```dart // building our own widget: class CustomCommand extends Widget { @override Widget generate(Context context){ // using an existing widget return Command('custom') } } ```
